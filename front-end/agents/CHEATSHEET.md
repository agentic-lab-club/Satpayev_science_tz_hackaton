# ⚡ TZ·AI Chat - Краткая справка (Шпаргалка)

## 🎯 3 минуты на успех

### 1️⃣ Запустить
```bash
npm run dev
# Откроется http://localhost:3000
```

### 2️⃣ Перейти на чат
```
http://localhost:3000/chat
```

### 3️⃣ Выбрать пример или написать вопрос

---

## 📋 Функции

| Функция | Как использовать |
|---------|-----------------|
| **Выбрать пример** | Клик на карточку с примером запроса |
| **Написать вопрос** | Введите текст + Enter |
| **Новая строка** | Shift + Enter |
| **Очистить диалог** | Кнопка "🗑️ Очистить диалог" |
| **Полный чат** | Со страницы дашборда: кнопка "↗" |
