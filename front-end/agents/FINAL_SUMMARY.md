## 🚀 Страницы приложения

### Home (/)
- Лендинг с описанием
- Кнопки "Начать анализ" и "Консультация ИИ"
- Навигация на оба раздела

### Dashboard (/dashboard)
- Загрузка и анализ ТЗ
- Список документов
- Статистика
- ChatWidget с ссылкой на /chat

### Chat (/chat) **[НОВОЕ]**
- Полный интерфейс чата
- 6 примеров запросов
- История сообщений
- Кнопка очистки диалога

---

## 💻 Технологический стек

- **Framework:** Next.js 13+ с App Router
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS
- **State Management:** React Hooks (useState, useRef, useEffect)
- **Components:** Functional components с "use client"
