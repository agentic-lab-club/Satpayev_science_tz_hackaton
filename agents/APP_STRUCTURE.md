# TZ·AI - Структура приложения

## Навигация

```
/ (Home)
├── /dashboard         # Дашборд с загрузкой и анализом ТЗ
└── /chat             # Полноценный интерфейс AI-ассистента
```

## Страницы

### `/` (Home page)
**Назначение:** Лендинг и главная точка входа
**Компоненты:**
- Header с ссылками на основные разделы
- Hero секция с описанием сервиса
- Карточки функций (Дашборд, AI-Ассистент)
- Кнопки CTA (Call-to-Action)
- Footer

**Навигация на странице:**
- Ссылка "📊 Дашборд" → `/dashboard`
- Ссылка "💬 Чат" → `/chat`
- Кнопка "Начать анализ ТЗ" → `/dashboard`
- Кнопка "Консультация ИИ" → `/chat`

### `/dashboard`
**Назначение:** Основной рабочий интерфейс для анализа ТЗ
**Компоненты:**
- `DashboardHeader` - заголовок с ссылкой на чат
- `StatCard` × 4 - статистика (документы, проанализировано, средний балл, проблемы)
- `DocumentList` - список загруженных документов с фильтрацией
- `QuickTips` - советы для улучшения качества
- `ChatWidget` - плавающее окно чата с кнопкой → `/chat`
- `UploadModal` - модальное окно загрузки файлов

**Новые элементы:**
- Кнопка "💬 Ассистент" в header → `/chat`
- Кнопка "↗" в ChatWidget для открытия полного чата → `/chat`

### `/chat`
**Назначение:** Расширенный интерфейс AI-ассистента для консультаций
**Компоненты:**
- `ChatHeader` - заголовок с информацией об ассистенте и счетчиком сообщений
- `ExamplePrompts` - система подсказок с 6 готовыми примерами запросов
- `MessageList` - история сообщений с анимацией
- `ChatInput` - поле ввода с поддержкой многострочности + **кнопка очистки диалога**

**Функциональность:**
- 6 примеров запросов в 2 категориях:
  - 📖 Правила составления ТЗ (4 примера)
  - ⚡ Улучшение текущего ТЗ (2 примера)
- Имитация ответа ассистента (случайный выбор из 4 вариантов)
- Кнопка "🗑️ Очистить диалог" с подтверждением
- Auto-scroll к новым сообщениям
- Индикатор статуса ("обработка запроса" / "онлайн")

## Цветовая схема (единая)

- **Background:** `#080d14` (primary dark)
- **Secondary:** `#0f1419` (slightly lighter)
- **Accent:** `amber-400` (primary interactive color)
- **Text:** `white` (primary), `slate-400`/`slate-500` (secondary)
- **Borders:** `slate-700` (default), `amber-400/30` (hover/focus)

## Анимации (единые)

- **Fade-in:** 300ms для новых сообщений
- **Transitions:** 200-300ms для hover эффектов
- **Spin:** на кнопке при загрузке
- **Bounce:** на индикаторе печати
