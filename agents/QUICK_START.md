# 🚀 Quick Start Guide - TZ·AI

## Что было создано?

Полнофункциональное web-приложение для анализа технических заданий с помощью ИИ.

**✅ Все 4 фазы разработки завершены**

---

## 🎯 Быстрый старт

### 1. Установка и запуск

```bash
# Перейти в директорию проекта
cd /Users/ilassalimov/Projects/satpayev

# Установить зависимости (если еще не установлены)
npm install

# Запустить dev сервер
npm run dev
```

Приложение откроется на `http://localhost:3000`

### 2. Доступные страницы

| Страница | URL | Что там |
|----------|-----|---------|
| 🏠 Главная | `http://localhost:3000/` | Landing page с навигацией |
| 📊 Dashboard | `http://localhost:3000/dashboard` | Загрузка и анализ ТЗ |
| 💬 Chat | `http://localhost:3000/chat` | AI-ассистент с примерами |
| 🔐 Auth | `http://localhost:3000/auth` | Вход / Регистрация |

---

## 📱 Что можно делать на каждой странице

### Главная страница (/)
- Просмотр описания проекта
- Переход на Dashboard или Chat
- Навигация на Auth страницу

### Dashboard (/dashboard)
- 📤 Загружать файлы с ТЗ (drag-and-drop)
- 📋 Видеть историю анализов
- 🔍 Искать документы
- 💬 Открыть Chat ассистента

### Chat (/chat)
- 💬 Общаться с AI-ассистентом
- 📚 Использовать 6 встроенных примеров
- 🗑️ Очищать историю (с подтверждением)
- ⌨️ Shift+Enter для новой строки
- 📊 Видеть счетчик символов

### Auth (/auth)
- 👤 Переключаться между Login и Register
- 📧 Вводить email
- 🔑 Вводить пароль
- 💾 Нажимать Remember me (Login only)
- 🔵 Кнопки социальной авторизации (Google, GitHub)

## 📖 Примеры использования

### Пример 1: Загрузка файла
1. Перейти на `/dashboard`
2. Нажать "Загрузить ТЗ"
3. Выбрать проект
4. Перетащить файл или нажать "Выбрать файл"

### Пример 2: Использование Chat
1. Перейти на `/chat`
2. Увидеть 6 примеров вопросов
3. Нажать на один из примеров
4. Или написать свой вопрос
5. Нажать кнопку "Отправить" (или Enter)

### Пример 3: Авторизация
1. Перейти на `/auth`
2. Увидеть форму Login
3. Нажать "Зарегистрируйтесь" для регистрации
4. Заполнить форму
5. Нажать "Войти" или "Зарегистрироваться"

## ❌ Известные ограничения

Текущие ограничения (требуют backend):

1. **Upload** - файлы не сохраняются (нужен backend)
2. **Chat** - ответы не идут от AI (заглушка)
3. **Auth** - регистрация не работает (нужен backend)

**Это нормально для фронтенд-части!**
