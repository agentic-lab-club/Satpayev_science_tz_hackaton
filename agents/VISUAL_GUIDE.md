## 🎬 Анимации

### 1. Fade-in для новых сообщений

```
Шаг 1 (0ms):
┌──────────┐
│ opacity  │ = 0
│ transform│ = translateY(10px)
│ Message  │ (невидимо внизу)
└──────────┘

Шаг 2 (150ms):
┌──────────┐
│ opacity  │ = 0.5
│ transform│ = translateY(5px)
│ Message  │ (полупрозрачно)
└──────────┘

Шаг 3 (300ms):
┌──────────┐
│ opacity  │ = 1
│ transform│ = translateY(0)
│ Message  │ (полностью видно)
└──────────┘
```

### 2. Bounce для индикатора печати

```
Точка 1:      Точка 2:          Точка 3:
   ↑            ↑ ↑                ↑ ↑ ↑
●●●    →    ●●●      →    ●●●
(вверх)    (разные      (вверх)
           времена)
```

### 3. Spin для кнопки отправки

```
  ↑          →      ↓       ←       ↑
  │        / \      │       \      │
  │       |   |     │        |    |   |
```
