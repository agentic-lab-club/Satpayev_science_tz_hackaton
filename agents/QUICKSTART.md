# 🚀 Быстрый старт - Chat Assistant

## Структура проекта

```
TZ·AI
├── Home page (/) - Лендинг
├── Dashboard (/dashboard) - Загрузка и анализ ТЗ
└── Chat (/chat) - AI-ассистент для консультаций
```

## Использование

### 1️⃣ Запуск приложения

```bash
cd /Users/ilassalimov/Projects/satpayev
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

### 2️⃣ Навигация

**Главная страница (/):**
- Кнопка "Начать анализ ТЗ" → Дашборд
- Кнопка "Консультация ИИ" → Чат
- Header: "📊 Дашборд" / "💬 Чат"

**Дашборд (/dashboard):**
- Кнопка "💬 Ассистент" (header) → Чат
- Кнопка "↗" (ChatWidget) → Чат (полный интерфейс)

**Чат (/chat):**
- Полноценный интерфейс AI-ассистента
- Примеры запросов + история сообщений
- Кнопка "🗑️ Очистить диалог" для сброса

### 3️⃣ Работа с чатом

#### Способ 1: Выбрать пример
1. На странице `/chat` видите 6 готовых примеров
2. Кликните на любой → отправляется как обычное сообщение
3. Получите ответ от AI

#### Способ 2: Написать свой вопрос
1. Введите текст в поле ввода
2. Нажмите `Enter` или кнопку `↑`
3. Дождитесь ответа (индикатор печати - 3 точки)

#### Способ 3: Многострочное сообщение
1. Введите текст, нажимайте `Shift+Enter` для новой строки
2. Когда готово, нажмите `Enter` для отправки

#### Очистить диалог
1. Нажмите кнопку "🗑️ Очистить диалог"
2. Подтвердите в диалоге
3. История сообщений будет удалена, чат начнется заново

## 🎯 Функциональность

### ✨ История сообщений
- Автоматически загружаются при открытии страницы
- Разные цвета для ваших (желтые) и ответов ассистента (серые)
- Плавная анимация появления новых сообщений

### 💡 Система подсказок
- Видны только при загрузке страницы (пока нет сообщений)
- Исчезают после первого отправленного вопроса
- Можно вернуться через очистку диалога

### 🤖 AI-ассистент
- Имитирует ответ с задержкой ~1.5 сек
- Показывает индикатор печати (3 прыгающие точки)
- Выбирает случайный ответ из базы (4 варианта)
- Готов для интеграции с реальным API

### 🗑️ Очистка диалога
- Удаляет всю историю сообщений
- Возвращает к начальному состоянию
- Требует подтверждение ("Вы уверены?")

- Fade-in (300ms) для новых сообщений
